# MyMentor
Mobile application project designed for CSC 4330. MyMentor fosters professional, yet personal relationships between 
students and teachers. These relationships will help students throughout their college journey and beyond. MyMentor 
provides a platform that teachers can use to help mentor students in both their inter- and extra-curriculum needs.

MyMentor consists of:
- Xamarin solution
  - Xamarin core project
  - Xamarin Android
  - Xamarin iOS
- .NET back-end solution
  - API
  - DataServices
  
Both of these solutions can be loaded and run in Microsoft's Visual Studio IDE.
